# Code of Conduct

This project expects direct, respectful, technical collaboration. The goal is to
make the portable setup easier to understand, review, and safely adapt.

Contributors should focus criticism on code, docs, claims, and evidence. Do not
use personal attacks, harassment, threats, discrimination, or private
information to pressure another person.

Maintainers may edit, hide, or remove comments and may close issues or restrict
participation when conduct gets in the way of useful project work.

If you need to report conduct privately, contact the maintainer through the
channels listed on their GitHub profile.

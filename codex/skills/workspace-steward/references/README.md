# Workspace Steward References

Reference material for the `workspace-steward` skill.

- `project-template/` is a ready-to-clone umbrella workspace skeleton. Copy its
  contents into a new project root to start with the clean main, PR worktree,
  experiments, local docs, artifacts, tmp, and archived layout the skill
  prescribes. Each directory ships a short README that states its purpose and
  rules.

# experiments

Exploratory work that is not yet a repo, an artifact, or a PR: spikes, drafts,
and prototypes whose home is not decided.

Promote work out of here when it earns a place: a repo checkout at the root, a
PR worktree under `../worktrees/prs/`, evidence under `../artifacts/`, or a doc
under `../docs/`. Long-lived experiments are a signal that they should have been
promoted already.
